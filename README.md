# mosya（模写コーディング集）

CodeJump などを使って行った **模写コーディングの成果物**をまとめるリポジトリです。  
「学んだことを手を動かして形に残す」ことを目的に、1つずつ積み上げていきます。

---

## 収録作品

### 1) Profile（CodeJump）
- お手本（CodeJumpデモ）：https://code-jump.com/demo/html/profile/
- コード：`/profile`
- ローカル表示：`profile/index.html`

> GitHub Pages を有効にしたら、ここにデモURLを追記します  
> 例：`https://oni-ch-dev.github.io/mosya/profile/`

---

## ディレクトリ構成

- `profile/`
  - `index.html`
  - `css/`
  - `img/`

（※ 模写を増やす場合は、同じ階層にフォルダを追加していく想定です）

---

## ローカルでの見方

### もっとも簡単（ファイルを直接開く）
1. リポジトリをクローン
2. `profile/index.html` をブラウザで開く

### VS Code の Live Server を使う（推奨）
- VS Code 拡張機能「Live Server」を入れて `index.html` を右クリック → **Open with Live Server**

---

## 新しい模写を追加する手順（運用ルール）

1. 作品ごとにフォルダを作成（例：`landing-page/`）
2. フォルダ直下に `index.html` を置く  
3. CSS・画像はそのフォルダ内に閉じる（例：`landing-page/css`, `landing-page/img`）
4. この README の「収録作品」に、参照元URLとフォルダ名を追記する

---

## 目的

- HTML/CSS の基礎定着（レイアウト・余白・レスポンシブ）
- 実装の“型”を身につける
- 成果物として履歴を残す（note の制作日記とも連携する）
